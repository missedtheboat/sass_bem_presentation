Functions


@import "bourbon";

@function shade($color, $percent) {
  @return mix(#000, $color, $percent);
}

@function tint($color, $percent) {
  @return mix(#fff, $color, $percent);
}
  • are conceptually similar to mixins
  • accept arguments
  • output is a single value
  • output can be any Sass data type [numbers, strings, colors, booleans, lists, maps]

CSS Architecture
"Object oriented CSS (OOCSS) is a methodology of writing maintainable, reusable, and scalable CSS
OOCSS Principles
  1. Separate structure from skin

    Positioning, (position, float, margin), separated from styling, things like (background-color, border, color).

  2. Separate container from content

    Suggests a styled element should never be dependent on where it is at in a page. Allows for styling reuse without DOM dependency.

Block Element Modifier
What is BEM
What is BEM
BEM is simple yet powerful naming methodology to make front-end code less confusing and brittle, more maintainable, easier to scale, and a lot more strict.
What is BEM
  • No external dependencies
  • Nothing required aside form BEM naming methodology
  • Independent blocks && css selectors make code reusable and modular
  • Methodology is a tool - reconfigure to your liking
BEM is a naming methodology 

.BLOCK{__ELEMENT[--MODIFIER]}
  • Block - an object or feature (search form, menu bar)
  • Element - the Block's children (search input, menu bar link)
  • Modifier
    • State of an element or block - (search active, menu bar link disabled)
    • A general modifier - cancel button, menu bar dark, menu bar light

var UserSearchView = (function() {

  var handleSearch = function(event, query) {
    EventAggregator.trigger('userSearch::query', {
      query: query
    });
  };

  return {
    initialize: function() {
      this.$el = $('.user-search');
      this.$el
        .find('.search__button')
        .click(handleSearch);
    },

    disable: function() {
      this.$el.addClass('search--is-disabled');
    },

    enable: function() {
      this.$el.removeClass('search--is-disabled');
    }
  };
})();
